## Live Table

A demo app showcasing a state aware live table using almost no JavaScript

- Tech
  - Golang (Fiber)
  - Templ
  - HTMX
  - Sqlite
  - Websockets
  - Webcomponents

---

### How to run

```sh
# install deps
go install
```

```sh
# run app in dev mode
go run .
```

```sh
# build and run app
go build -o app .
./app
```
